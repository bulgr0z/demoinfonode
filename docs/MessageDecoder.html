    <private> _decodeRawMessages(packetBuffer, $decoded)

    Decodes 0 or more protobuf messages sliced from the DemoBuffer (ie. the packet's "raw data"). each message starts with a header of 2 varint32 : - command (corresponding to a protobuf message) - message length Varints being of `variable` size, we need to keep track of the header size to properly extract the message
    Parameters:
    Name Type Description
    packetBuffer Buffer DemoBuffer slice (raw part of the packet)
    $decoded Promise Promise obj to resolve when decoded
